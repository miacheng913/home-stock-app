# 家裡有多少

一個可以安裝到手機主畫面的家庭用品庫存 PWA。

## 功能

- 新增、編輯與刪除用品
- 快速增加／減少庫存
- 自訂單位、分類、最低庫存、位置與備註
- 低庫存自動加入採購清單
- 採購勾選與「已補貨」快速更新
- 搜尋與分類篩選
- 資料保存在裝置的瀏覽器中
- 安裝後可離線使用

## 本機啟動

請用 HTTP 伺服器開啟資料夾，不要直接雙擊 `index.html`，否則離線快取無法啟用。

若有 Python：

```powershell
python -m http.server 4173 --directory .
```

然後打開 `http://localhost:4173`。

## 安裝到手機

- Android Chrome：開啟網址後選「安裝應用程式」或「加到主畫面」。
- iPhone Safari：點分享按鈕，再選「加入主畫面」。

目前資料只存在使用中的裝置與瀏覽器。清除瀏覽器網站資料會一併刪除庫存記錄。
